# -*- coding: utf-8 -*-
CreatePlugin is a plugin for spreads (http://github.com/DIYBookScanner/spreads)

Install:
$ sudo pip install git+http://github.com/DIYBookScanner/spreadsplug_createplugin.git

Configure:
# this is currently quite unfortunate, you need to configure the plugin before using it
$ spread configure
# select CreatePlugin from the list of plugins

Use:
$ spread createplugin <SpreadsPlugin,DevicePlugin,HookPlugin> MyNewPlugin
